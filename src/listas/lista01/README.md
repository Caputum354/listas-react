# Lista 01 — Exercícios básicos de ReactJS (Iniciante)

Tema: **variáveis e interpolação `{}` no JSX**.

1. Mostrar uma variável na tela — "Olá, meu nome é João"
2. Mostrar idade — "Eu tenho 25 anos"
3. Soma de dois números — "A soma é: 15"
4. Mostrar cidade e país — "Eu moro em São Paulo - Brasil"
5. Variável booleana — "Estou estudando React: true"
6. Informações de uma pessoa (nome, idade, profissão)
7. Preço de produto (produto, preco)
8. Variável de saudação — "Bom dia, Carlos!"
9. Multiplicação — "Multiplicação: 40"
10. Ano atual — "Estamos no ano de 2026"
11. Frase com 3 variáveis (nome, curso, escola)
12. Idade aproximada (anoAtual - anoNascimento)
13. Número favorito
14. Temperatura — "A temperatura em Rio de Janeiro é 28°C"
15. Mini perfil (nome, idade, hobby)
